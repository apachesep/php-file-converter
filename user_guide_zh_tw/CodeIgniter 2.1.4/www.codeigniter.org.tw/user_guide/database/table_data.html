資料表資訊

底下函式將帶您取得資料表資訊。

$this->db->list_tables();

回傳陣列值其中包含您所聯接的資料庫全部資料表名稱。參考範例:

$tables = $this->db->list_tables();

foreach ($tables as $table)
{
   echo $table;
}

$this->db->table_exists();

針對您所需要操作的資料表，可以利用底下函式判斷資料表示否存在。回傳布林函數值 TRUE/FALSE. 使用範例:

if ($this->db->table_exists('table_name'))
{
   // some code...
}

注意: 請將 table_name 取代成您所需要操作的資料表。
