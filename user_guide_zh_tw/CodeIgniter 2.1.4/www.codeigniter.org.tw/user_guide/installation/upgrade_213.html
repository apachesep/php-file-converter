從 2.1.2 升級到 2.1.3

在執行升級之前，你可以透過更換 index.php 使得網站狀態為 offline。

步驟 1: 更新網站 CodeIgniter 檔案

取代 system 目錄底下的全部檔案及及根目錄 index.php 檔案。假如您有更動到 index.php，請麻煩自行更新。

注意: 假如您有修改此目錄底下檔案，麻煩請先複製備份。
