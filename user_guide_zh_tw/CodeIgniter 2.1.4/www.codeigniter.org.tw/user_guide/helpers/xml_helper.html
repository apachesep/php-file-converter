XML 輔助函數

XML 輔助函數裡所包含的函數是用來處理 XML 資料。

載入 輔助函數

載入輔助函數所使用的程式碼:

$this->load->helper('xml');

底下的函數就可以使用了:

xml_convert('string')

取得字串當作是輸入(input)，並轉換成(convert)底下所保留的 XML 字元至實體(entitles):

Ampersands: &
小於以及大於符號: < >
單引號以及雙引號: '  "
減號: -

此函數略過 ampersands (&)，假如那是現有字元實體(character entities)的一部分。參考範例:

$string = xml_convert($string);
